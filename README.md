# 🔐 Sistema de Login com Node.js + MySQL

Projeto completo de autenticação de usuários com back-end em Node.js. Ideal para estudo ou uso como base em sistemas reais.

## 🚀 Funcionalidades
- Cadastro de usuário
- Login com autenticação segura (bcrypt)
- Mensagem de erro para usuários não encontrados
- Sessão com `express-session`
- Redirecionamento após login
- Visual moderno com HTML + CSS

## 🛠️ Tecnologias utilizadas
- Node.js
- Express
- MySQL
- Express-session
- Bcryptjs
- Dotenv
- EJS (pode trocar por HTML se quiser)
